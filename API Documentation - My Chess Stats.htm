My Chess Stats API

Base URL: http://localhost:8000/api/

All API endpoints require authentication. The API uses Django's session authentication with CSRF protection.

Authentication

Session Authentication

The API uses Django's session authentication. Include the session cookie and CSRF token in your requests.

Login Endpoint
POST /login/
Content-Type: application/json

{
    "username": "yourusername",
    "password": "yourpassword"
}
CSRF Token

For POST, PUT, DELETE requests, include the CSRF token:

// JavaScript function to get CSRF token
function getCookie(name) {
    let cookieValue = null;
    if (document.cookie && document.cookie !== '') {
        const cookies = document.cookie.split(';');
        for (let i = 0; i < cookies.length; i++) {
            const cookie = cookies[i].trim();
            if (cookie.substring(0, name.length + 1) === (name + '=')) {
                cookieValue = decodeURIComponent(cookie.substring(name.length + 1));
                break;
            }
        }
    }
    return cookieValue;
}

// Include in request headers
headers: {
    'X-CSRFToken': getCookie('csrftoken'),
    'Content-Type': 'application/json'
}

List Games

GET /api/games/

Retrieves a paginated list of the authenticated user's games.

Query Parameters
Parameter Type Description Example
page integer Page number for pagination page=2
platform string Filter by platform (chesscom/lichess) platform=chesscom
result string Filter by result (W/L/D) result=W
time_class string Filter by time control time_class=blitz
search string Search opponent name search=magnus
order_by string Sort field (+/- prefix for direction) order_by=-date_played
Example Request
GET /api/games/?platform=chesscom&result=W&order_by=-date_played&page=1

Import Games

POST /api/games/import/

Bulk import games from Chess.com CSV or Lichess PGN files.

Request Format: multipart/form-data
Field Type Required Description
file file Required CSV or PGN file
platform string Required 'chesscom' or 'lichess'
username string Required for Lichess Your username on the platform
Example Request (Chess.com)
POST /api/games/import/
Content-Type: multipart/form-data

file: @games_export.csv
platform: chesscom
Example Request (Lichess)
POST /api/games/import/
Content-Type: multipart/form-data

file: @lichess_games.pgn
platform: lichess
username: yourlichessusername
Example Response (201 Created)
{
    "imported_count": 25,

Error Handling

HTTP Status Codes
Code Description Example
200 OK Request successful
201 Created Resource created successfully
204 No Content Successful delete
400 Bad Request Missing required fields, validation errors
401 Unauthorized Authentication required
403 Forbidden CSRF token missing/invalid
404 Not Found Game not found
500 Internal Server Error Server-side error
Error Response Format
{
    "error": "Error message",
    "field_name": ["Specific field error"]
}
Example Validation Error
{
    "my_rating": ["Rating must be between 0 and 3500"],
    "date_played": ["This field is required"]
}
